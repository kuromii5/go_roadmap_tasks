# CI/CD (GitLab & Kubernetes)

## Что изучаем
- **Облачная инфраструктура:** развертывание связки из GitLab и Managed Kubernetes в облаке (например, Yandex Cloud) для создания полноценной среды разработки.
- **Инструменты автоматизации:** установка **GitLab Runner** и **GitLab Agent** через маркетплейс для управления ресурсами кластера напрямую из репозитория.
- **Сборка образов:** использование **Kaniko** для быстрой и безопасной сборки Docker-контейнеров внутри Kubernetes-native среды.
- **Управление деплоем:** настройка пайплайнов через `.gitlab-ci.yml`, написание YAML-манифестов и решение проблем с доступом к приватным реестрам через **ImagePullSecrets**.
