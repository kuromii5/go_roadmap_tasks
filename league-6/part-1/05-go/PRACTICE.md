# Практика — Консольный магазин

Ты будешь строить консольный магазин шаг за шагом. Каждый шаг добавляет новую возможность к тому что уже написано — в итоге получится одна рабочая программа.

---

## Шаг 1 — Переменные и вывод

Создай переменные для одного товара и выведи информацию о нём.

- Название товара — `string`
- Цена — `float64`
- Количество на складе — `int`
- Есть ли в наличии — `bool`

Выведи:
```
Товар: Ноутбук
Цена: 80000.00 руб.
На складе: 5 шт.
В наличии: true
```

---

## Шаг 2 — Условия

Добавь к программе логику скидок.

- Если цена больше 50000 → скидка 10%, вывести `"Применена скидка 10%"`
- Если цена от 10000 до 50000 → скидка 5%, вывести `"Применена скидка 5%"`
- Иначе → `"Скидка не применяется"`

После скидки выведи новую цену.

---

## Шаг 3 — Цикл

Теперь у тебя не один товар, а несколько. Объяви три товара с разными ценами и количеством.

- В цикле выведи каждый товар с его номером (1, 2, 3)
- Посчитай и выведи общую стоимость всех товаров на складе (`цена × количество`)

```
1. Ноутбук     — 80000.00 руб. x 5 шт. = 400000.00 руб.
2. Мышь        —  1500.00 руб. x 3 шт. =   4500.00 руб.
3. Монитор     — 30000.00 руб. x 1 шт. =  30000.00 руб.
---
Итого на складе: 434500.00 руб.
```

---

## Шаг 4 — Функции

Вынеси повторяющиеся действия в функции:

- `applyDiscount(price float64, percent float64) float64` — возвращает цену после скидки
- `totalCost(price float64, quantity int) float64` — возвращает стоимость позиции
- `printItem(number int, name string, price float64, quantity int)` — выводит строку товара

Перепиши шаг 3 используя эти функции.

---

## Шаг 5 — Указатели

Добавь функцию `increasePrice(price *float64, amount float64)` которая увеличивает цену товара на заданную сумму, изменяя оригинальную переменную.

Вызови её для первого товара, увеличь цену на 5000, выведи цену до и после.

Проговори для себя — почему без указателя это не сработает.

---

## Шаг 6 — Структура

Сейчас у тебя куча отдельных переменных на каждый товар. Замени их структурой:

```go
type Product struct {
    Name     string
    Price    float64
    Quantity int
}
```

- Перенеси все товары в `[]Product`
- Перепиши функции из шага 4 чтобы они принимали `Product`
- Метод `Total() float64` — добавь к структуре вместо отдельной функции `totalCost`
- Метод `ApplyDiscount(percent float64)` — меняет цену внутри структуры

---

## Шаг 7 — defer

Оберни всю программу в функцию `runShop()`. В начале `runShop` добавь через `defer` вывод:

```
Магазин закрыт. Спасибо за работу!
```

Убедись что эта строка выводится последней — после всех товаров.

---

## Шаг 8 — Слайсы и мапы

Добавь к программе аналитику:

- Посчитай через `map[string]float64` общую стоимость по каждому товару (`name → total`)
- Найди самый дорогой товар (по цене за штуку)
- Найди товар с максимальным запасом на складе

Выведи итоги аналитики отдельным блоком после списка товаров.

---

## Шаг 9 — Пакеты

Раздели программу на пакеты:

```
shop/
├── cmd/
│   └── main.go
├── internal/
│   └── product/
│       └── product.go
└── go.mod
```

- Структура `Product` и все её методы — в `internal/product`
- Аналитика — туда же, отдельной функцией `Analyze(products []Product)`
- `main.go` — только запуск

---

## Шаг 10 — Консольный ввод (итог)

Сделай программу интерактивной. Она работает в бесконечном цикле и принимает команды:

```
list                        → список всех товаров
add Клавиатура 3000 10      → добавить товар (название, цена, количество)
discount 1 15               → применить скидку 15% к товару #1
restock 2 5                 → добавить 5 штук товара #2
total                       → общая стоимость склада
exit                        → завершить
```

Ввод через `bufio.Scanner`, разбор через `strings.Fields`.
